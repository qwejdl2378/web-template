# Usage
## Install gonew cli
```shell
go install golang.org/x/tools/cmd/gonew@latest
```

## Generate new project
```shell
# eg: cd $GOPATH/src
cd ${your/workspace}

# eg: gonew github.com/qwejdl2378/web-template github.com/qwejdl2378/echo
gonew github.com/qwejdl2378/web-template  ${your/module/name}

# eg: cd echo
cd ${your/module/name}
```
